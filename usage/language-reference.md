# ForgeScript 言語リファレンス

ForgeScript v0.1.0 (MVP) の文法と機能の詳細仕様です。

## 概要

ForgeScript は Rust の所有権・ライフタイム・可変性を抽象化したスクリプト言語です。Cargo エコシステムとの統合を目指しています。現バージョン (MVP) では、変数宣言、四則演算、文字列連結、ネイティブ関数呼び出しをサポートしています。

## 字句要素

### 識別子

- 英字または `_` で始まる
- 英字、数字、`_` の組み合わせ
- 例: `x`, `my_var`, `_private`

### キーワード

| キーワード | 用途 |
|-----------|------|
| `let` | 変数宣言 |

### リテラル

**整数リテラル:**
- 10進数のみ
- 例: `0`, `42`, `100`

**文字列リテラル:**
- ダブルクォート `"` で囲む
- 例: `"Hello"`, `"Hello, World"`
- MVP ではエスケープシーケンスは未対応

### 演算子

| 演算子 | 意味 | 優先度 |
|-------|------|--------|
| `+` | 加算 / 文字列連結 | 低 |
| `-` | 減算 | 低 |
| `*` | 乗算 | 高 |
| `/` | 除算 | 高 |
| `=` | 代入（`let` と併用） | — |
| `(` `)` | グループ化 | 最高 |

### 区切り記号

- `(` `)` — 関数呼び出し、式のグループ化
- `,` — 関数引数の区切り

## 文 (Statements)

### 変数宣言 (`let`)

```fs
let <識別子> = <式>
```

- 変数はグローバルスコープ
- 宣言時に必ず初期値を指定
- 再代入は不可（イミュータブル）

**例:**

```fs
let x = 1
let name = "Alice"
let result = 2 + 3 * 4
```

### 式文 (Expression Statement)

```fs
<式>
```

式単体も文として書けます。主に副作用（`print` 呼び出しなど）に使用します。

```fs
print(42)
print(x + y)
1 + 2
```

## 式 (Expressions)

### リテラル

```fs
42
"Hello"
```

### 識別子（変数参照）

```fs
x
my_var
```

### 二項演算

```fs
<式> <演算子> <式>
```

**算術演算:**

```fs
1 + 2
10 - 3
4 * 5
20 / 4
```

**文字列連結:**

```fs
"Hello" + " " + "World"
```

### 演算子の優先順位

1. `*` `/` （乗算・除算）— 高
2. `+` `-` （加算・減算）— 低
3. 同じ優先度は左から右に評価

**例:**

```fs
2 + 3 * 4    // => 14 (3*4 が先)
(2 + 3) * 4  // => 20
10 - 3 - 2   // => 5 (左から)
```

### 関数呼び出し

```fs
<識別子>(<引数>, ...)
```

**例:**

```fs
print(42)
print("Hello")
print(x + y)
```

**引数:**
- 0個以上
- 式をカンマ区切りで指定
- ネイティブ関数 `print` は任意の値を1つ受け取る

## 型

MVP でサポートする型:

| 型 | 説明 | リテラル例 |
|----|------|-----------|
| 整数 (Int) | 符号付き整数 | `42` |
| 文字列 (String) | UTF-8 文字列 | `"Hello"` |
| Nil | 値なし | （リテラルなし） |

## ネイティブ関数

### `print(value)`

- **説明**: 値を標準出力に出力し、改行する
- **引数**: 1つ（整数、文字列、その他）
- **戻り値**: なし (Nil)

```fs
print(42)
print("Hello")
print(x)
```

## スコープと変数の可視性

- すべての変数は**グローバルスコープ**
- 宣言前に参照するとコンパイルエラー
- 同じ名前の再宣言は不可

## 未サポート機能 (MVP)

以下の機能は現バージョンでは**サポートされていません**:

- 関数定義
- 制御構造 (`if`, `while`, `for`)
- 配列・リスト
- オブジェクト・構造体
- モジュール・インポート
- コメント
- 浮動小数点数
- 真偽値と比較演算子 (`<`, `>`, `==` など)
- 論理演算子 (`&&`, `||`, `!`)
- 単項演算子（`-x` など）
- 1行複数文 (`;` 区切り)
- エスケープシーケンス（`\n`, `\t` など）

## ファイル形式

- **推奨拡張子**: `.fs`
- **エンコーディング**: UTF-8
- **改行**: LF または CRLF
